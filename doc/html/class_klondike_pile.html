bool KlondikePile::IsValid ( const Card & card)
inlineoverridevirtual

Checks if a card can be legally added to this pile based on Klondike Solitaire rules.

Parameters
cardThe card to check for validity.
Returns
true if the card can be added, false otherwise.

A card can be added to the pile if:

  • The pile is empty and the card is a KING.
  • The pile is non-empty, and the card is of opposite color and one rank lower than the top card.
